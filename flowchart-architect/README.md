# Flowchart Architect

A comprehensive system for creating, managing, and optimizing business process flowcharts using specialized agents and skills.

## Overview

Flowchart Architect provides an end-to-end solution for business process documentation and visualization. It combines intelligent agents with specialized skills to deliver high-quality, standards-compliant flowcharts that meet stakeholder requirements.

## Key Components

### Agents
- **Asset Librarian**: Manages flowchart templates and components
- **Process Analyst**: Analyzes requirements and designs optimal solutions
- **XML Auditor**: Validates flowchart XML structures and compliance

### Skills
- **Solicit Requirements**: Gathers and clarifies stakeholder requirements
- **Analyze Gap**: Identifies gaps and improvement opportunities
- **Search Standard Assets**: Locates appropriate templates and components
- **Generate Draw.io**: Creates Draw.io-compatible flowchart XML
- **Orchestrate Process**: Coordinates end-to-end workflow

## Usage

### Basic Workflow
1. **Solicit Requirements** - Gather stakeholder needs and specifications
2. **Process Analysis** - Analyze requirements and design optimal flowchart structure
3. **Gap Analysis** - Identify gaps between current state and desired solution
4. **Asset Selection** - Choose appropriate templates and components
5. **Generation** - Create Draw.io-compatible flowchart XML
6. **Validation** - Ensure XML compliance and quality

### Advanced Features
- Template library management
- Custom styling and theming
- Batch generation capabilities
- Standards compliance checking
- Quality assurance validation

## Integration

Flowchart Architect integrates with:
- Claude Code for AI-powered assistance
- Draw.io for visual editing and collaboration
- organizational template libraries
- industry-standard flowchart notations

## Output Formats

- Draw.io XML files
- BPMN diagrams
- UML activity diagrams
- Cross-functional flowcharts
- Swimlane diagrams

## Quality Assurance

All generated flowcharts undergo rigorous validation:
- XML syntax and schema compliance
- Standards conformance checking
- Visual quality verification
- Stakeholder requirement validation

## Getting Started

1. Define your flowchart requirements
2. Use the Orchestrate Process skill for complete project management
3. Or use individual skills for specific tasks
4. Review and validate generated flowcharts
5. Export to your preferred format
