# Brand Guidelines for Flowcharts

Ensure all diagrams adhere to these visual standards.

## Color Palette

| Usage | Color Name | Hex Code | Description |
|-------|------------|----------|-------------|
| **Primary** | Corporate Blue | `#0056b3` | Main process steps, headers |
| **Secondary** | Action Orange | `#fd7e14` | Key actions, highlights |
| **Decision** | Logic Purple | `#6f42c1` | Decision diamonds |
| **Start/End** | Signal Green | `#28a745` | Start and End terminators |
| **System** | Tech Grey | `#6c757d` | System/Automated steps |
| **Error/Stop** | Alert Red | `#dc3545` | Error paths, stop events |

## Typography

- **Font Family**: Helvetica, Arial, or Sans-Serif
- **Title Font**: 18pt, Bold, Corporate Blue
- **Swimlane Header**: 14pt, Bold, Black
- **Step Text**: 12pt, Regular, Black
- **Label Text**: 10pt, Italic, Dark Grey

## Logo Usage

- Place the corporate logo in the **top-right corner** of the diagram.
- Maintain a minimum clear space of 20px around the logo.
- Do not distort or recolor the logo.

## Stroke & Line Styles

- **Process Flow**: Solid line, 1pt width, Black (`#000000`)
- **Data Flow**: Dashed line, 1pt width, Dark Grey (`#666666`)
- **Connectors**: Orthogonal (Right-angle) routing preferred.
