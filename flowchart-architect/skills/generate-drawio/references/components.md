# Component Library

Definitions of standard components available for generation.

## Standard Nodes

### `process_step`
- **Type**: `mxGraph.shape.rectangle`
- **Default Size**: 120x60
- **Style**: `rounded=1;whiteSpace=wrap;html=1;fillColor=#ffffff;strokeColor=#0056b3;fontColor=#000000;`

### `decision_node`
- **Type**: `mxGraph.shape.rhombus`
- **Default Size**: 80x80
- **Style**: `rhombus;whiteSpace=wrap;html=1;fillColor=#ffffff;strokeColor=#6f42c1;fontColor=#000000;`

### `terminator`
- **Type**: `mxGraph.shape.ellipse`
- **Default Size**: 80x40
- **Style**: `ellipse;whiteSpace=wrap;html=1;fillColor=#28a745;strokeColor=#28a745;fontColor=#ffffff;`

### `document`
- **Type**: `mxGraph.shape.document`
- **Default Size**: 100x50
- **Style**: `shape=document;whiteSpace=wrap;html=1;boundedLbl=1;fillColor=#ffffff;strokeColor=#000000;`
